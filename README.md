# SSM
A Smart Item System that keep track of your grocery so you don't need to.

# Goal 

SSM is a web application which stores all the grocery that you as a individual can store that you purchased and on when the items are about to be finish or about to expire and sends a email  to the registered email notifying you about the items.<br>

Another goal of this problem is to implement a recommandation algorithm that makes use of the items that you have purchased and make assoication of the products that you have bought and can recommend to the user. Although the algorithm is unique to seperate user who are using the web application.
